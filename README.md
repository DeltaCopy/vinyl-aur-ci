# vinyl-aur-ci

Scheduled pipeline (runs daily at ~12PM) to publish updated PKGBUILD for vinyl AUR packages.

Upstream: https://github.com/ekaaty/vinyl-theme

AUR package: https://aur.archlinux.org/packages/vinyl
